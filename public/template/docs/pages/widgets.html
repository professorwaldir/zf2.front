Widget Tabs

	<div class="widget-header">
	</div> <!-- /.widget-header -->
	
	<div class="widget-tabs">
		<ul class="nav nav-tabs">
		  <li class="active"><a href="#tab-1">Tab 1</a></li>
		  <li><a href="#tab-2">Tab 2</a></li>
		  <li><a href="#tab-3">Tab 3</a></li>
		</ul>		
	</div> <!-- /.widget-tabs -->
	
	<div class="widget-content">
	
		<div class="tab-content">

</div> <!-- /.widget -->

Widget Toolbars

NOTE: When adding buttons to toolbar make sure the you add the class .btn-small for correct sizing.

Top Toolbar

<div class="widget">
					
	<div class="widget-header">
		
	</div> <!-- /.widget-header -->
	
	<div class="widget-toolbar">
		<button class="btn btn-small">Primary Action</button>
	</div> <!-- /.widget-toolbar -->	
	
	<div class="widget-content">		
		
	</div> <!-- /.widget-content -->
	
</div> <!-- /.widget -->

Bottom Toolbar

To create a bottom toolbar move the .widget-toolbar div below .widget-content and add the class .toolbar-bottom to the widget container.

<div class="widget toolbar-bottom">
					
	<div class="widget-header">
		
	</div> <!-- /.widget-header -->	
	
	<div class="widget-content">		
		
	</div> <!-- /.widget-content -->
	
	<div class="widget-toolbar">
		<button class="btn btn-small">Primary Action</button>
	</div> <!-- /.widget-toolbar -->
	
</div> <!-- /.widget -->
