Tables

Table Options

Name Class Description
Default None No styles, just columns and rows
Basic .table Only horizontal lines between rows
Bordered .table-bordered Rounds corners and adds outer border
Zebra-stripe .table-striped Adds light gray background color to odd rows (1, 3, 5, etc)
Condensed .table-condensed Cuts vertical padding in half, from 8px to 4px, within all td and th elements

Table Highlight

For a little change in style add the class .table-highlight to your table tag for a dark blue table header and light blue table striping.

<table class="table table-highlight">

</table>

Widget Table

	<div class="widget-header">
		<h3>Widget Table</div>
	</div>
	
	<div class="widget-content">

</div>

Data Table Plugin

Includes

<link href="./js/plugins/datatables/DT_bootstrap.css" rel="stylesheet">
	
<script src="./js/plugins/datatables/jquery.dataTables.js"></script>
<script src="./js/plugins/datatables/DT_bootstrap.js"></script>

Helper

You can have the Data Plugin automatically attached by adding the following class to your tables: 

<table class="table data-table">

Javascript

$('.dataTable').dataTable ({
	sDom: "<'row'<'span6'l><'span6'f>r>t<'row'<'span6'i><'span6'p>>",
	sPaginationType: "bootstrap",
	oLanguage: {
		"sLengthMenu": "_MENU_ records per page"
	}
});

Responsive Table

To create a horizontally scrolling data table on mobile devices apply the following class to your table: 

<table class="table responsive">
